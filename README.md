# PC Console 项目管理看板

商务及运营同学日常工作进展自主录入与集成看板系统。

## 功能

- **进展录入**：支持一次性录入多条工作进展，项目/平台/类型可搜索下拉+自由输入
- **信息看板**：卡片式布局，按项目/平台/填写人筛选，统计总览
- **续录更新**：点击卡片续录后续进展，时间线展示历史更新

## 技术栈

- 前端：原生 HTML/CSS/JS（零依赖）
- 数据库：Supabase (PostgreSQL)
- 部署：Vercel + GitHub 自动部署

## 本地开发

```bash
# 启动本地服务器
python -m http.server 8090
# 访问 http://localhost:8090
```

## 部署

通过 GitHub 推送自动部署到 Vercel。
